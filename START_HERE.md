# 🎮 CodeVerse Adventure - START HERE

## ⚡ 30-Second Setup

```bash
# 1. Get API Key (2 min)
# Visit: https://rapidapi.com/judge0-official/api/judge0-ce
# Sign up → Subscribe → Copy key

# 2. Backend (Terminal 1)
cd backend
npm install
echo "JUDGE0_API_KEY=paste_your_key_here" > .env
npm run dev

# 3. Frontend (Terminal 2)
cd frontend
npm install
npm run dev

# 4. Play!
# Open: http://localhost:3000
```

**That's it! The game is running.** ✨

## ⚠️ Common Issues

**"CORS error?"**
→ Backend must be running on http://localhost:3001

**"Code won't execute?"**
→ Check backend console, verify Judge0 API key
